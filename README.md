## 项目主要技术栈

1、运用HTML5语义化标签+bootstrap进行页面布局。。 <br /> 
2、利用jQuery+jquery.singlePageNav.min.js进行导航栏的效果的跳转。 <br /> 
3、利用Wow.js+animate.css实现滚动页面的时候展示 CSS 动画 。 <br /> 

## 项目运行操作
download源码后直接双击运行index.html文件即可。 <br />
